Historia dos Computadores

/\ Voltar para o Inicio /\

A Primeira geração de computadores começou na epoca de 1944 com o Mark I.

Na epoca os computadores eram gigantes, usavam valvulas para o controle dos impusos eletricos com isso o ambiente que eles estavem eram quentes

A Temperatura alta atraia insetos com eles causando problema nos computadores, com isso foi popularizado o termo BUG que em portugues significa inseto

Os Computadores evoluiram muito com o passar das decadas, foram de computadores utilizando valvulas para transitores e depois para circuitos intregrados.

Estrutura Basica HTML

/\ Voltar para o Inicio /\

O HTML é estruturado em 2 partes. Uma é a Head e o outro e o Body

Head é tudo que o usuario não interage diretamente, nela fica informações sobre a pagina e links que a pagina usa

Por exemplo na tag head temos definição do idioma que a pagina está sendo construida, podemos tambem coloca a assinatura do autor, link do CSS que é utilizado na pagina

Body É tudo que o usuário do site ve. Nela está informações transmitidas para o usuario, links, imagens e campos de textos

Ela é a parte que o usuario interagem diretamente, sendo para apenas ler uma informação ou assistir um video, ate mesmo inserir informação
